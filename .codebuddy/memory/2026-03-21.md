# 2026-03-21

## 分析: vx 覆盖协作开发场景的改进需求

**分析对象**: AionUi (Electron+Bun+TypeScript) 和 AuroraView (Rust+Python+TypeScript)

**关键发现**:

### 缺失的 Provider（高优先级）
- **oxlint**: AionUi 核心 lint 工具，Rust 二进制，适合 github_rust_provider
- **ruff**: AuroraView 的 Python linter，Rust 二进制，适合 github_rust_provider  
- **maturin**: AuroraView 的构建核心(PyO3)，Rust 二进制

### 缺失的 Provider（中低优先级）
- **oxfmt**: JS/TS 格式化工具
- **nox**: Python 测试会话管理（可用 package_alias: `vx nox = vx uvx:nox`）
- **pdoc**: Python API 文档生成

### 功能改进建议
1. pre-commit/husky 适配统一（两个项目用不同的 hook 系统）
2. 项目模板扩充：Electron 模板、Rust+Python 模板
3. 新贡献者 onboarding：一行命令 bootstrap 脚本
4. CI 一键配置增强

### vx 已覆盖的核心能力
sync, lock, setup, hook, init, --with, fullstack 模板均已实现。
73 个 Provider 覆盖了大部分主流工具。AuroraView 已是 vx 深度用户。

## 实施: 新 Provider + init 模板 + project-analyzer 增强

### 新增 4 个 Provider
1. **oxlint** (`crates/vx-providers/oxlint/provider.star`)
   - 来源: oxc-project/oxc GitHub releases, tag前缀 `apps_v`
   - 含 bundled runtime: oxfmt（格式化工具）
   - 平台: windows/x64, windows/arm64, macos/x64, macos/arm64, linux/x64, linux/arm64
   
2. **ruff** (`crates/vx-providers/ruff/provider.star`)
   - 来源: astral-sh/ruff GitHub releases, 无 v 前缀 tag
   - Python 生态最快的 linter/formatter
   
3. **maturin** (`crates/vx-providers/maturin/provider.star`)
   - 来源: PyO3/maturin GitHub releases, v 前缀 tag
   - Rust+Python 混合项目的构建核心，Linux 用 musl

4. **openclaw** (`crates/vx-providers/openclaw/provider.star`)
   - 通过 npm 全局安装 (`npm install -g openclaw`)
   - 含 bundled runtime: clawhub（技能注册表 CLI）
   - 依赖 Node.js >= 22

### init 模板增强
- 新增 `electron` 模板: node@22 + pnpm + oxlint
- 新增 `rust-python` 模板: rust + python + uv + maturin + ruff
- 新增 `openclaw` 模板: node@22 + openclaw

### project-analyzer 增强
- Node.js rules: 新增 oxlint lint 规则(优先级100), oxfmt format 规则(优先级100)
- known_tools 列表: 添加 oxlint, oxfmt, biome, openclaw, clawhub, maturin
- auto-detect: oxlintrc.json, ruff.toml, Cargo.toml+pyo3, .openclaw/, SKILL.md

### Skills 文档更新
- skills/vx/SKILL.md: 工具列表从 50+ 更新到 75+，新增 JS Tooling 和 AI Agents 分类
- skills/vx/vx-project-setup.md: 添加新模板选项
- skills/vx/vx-commands-reference.md: init 命令展开模板列表
- crates/vx-cli/src/skills/vx-usage/SKILL.md: 同步更新工具列表

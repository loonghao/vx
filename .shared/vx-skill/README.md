# VX Skill - Universal Development Tool Manager

An AI skill that provides comprehensive knowledge about vx, the universal development tool manager for the AI era.

## Overview

This skill enables AI coding assistants to:
- Understand vx's architecture, commands, and usage patterns
- Help users set up development environments with vx
- Recommend appropriate tools and versions for projects
- Generate vx.toml configurations
- Troubleshoot vx-related issues

## Features

- **Complete Command Reference**: All vx commands with examples
- **Tool Knowledge**: Information about 50+ supported tools
- **Project Setup Patterns**: Best practices for different project types
- **Configuration Templates**: Ready-to-use vx.toml examples
- **Troubleshooting Guide**: Common issues and solutions

## Usage

The skill activates automatically when AI assistants detect vx-related queries. You can also invoke it explicitly:

```
How do I set up a Node.js project with vx?
Create a vx.toml for a Python ML project
What tools does vx support?
```

The skill includes comprehensive documentation about:
- vx architecture and design philosophy
- All CLI commands and options
- Supported tools and ecosystems
- Configuration file formats
